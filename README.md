# SistemaVentasRutas

Sistema de ventas en ruta con aplicacion Android offline, API REST en Spring Boot y despliegue posterior en Microsoft Azure.

## Estructura

- `docs/`: analisis, requisitos y decisiones de arquitectura.
- `backend/`: API REST y reglas de negocio.
- `android/`: aplicacion Android en Kotlin.

## Estado

El proyecto se encuentra en la fase de analisis funcional. El documento principal es [docs/analisis-funcional-v1.md](docs/analisis-funcional-v1.md).
